## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
# 测试环境打包 配置文件/src/config/global_test.js
npm run test

# build for production with minification
# 生产环境打包 配置文件/src/config/global_prod.js
npm run build
```
